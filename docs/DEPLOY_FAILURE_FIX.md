# 部署失败修复指南

## ❌ 部署失败原因

**错误信息**：
```
exec: "gunicorn": executable file not found in $PATH
```

**原因**：`requirements.txt` 中缺少 `gunicorn` 依赖包！

---

## ✅ 问题已修复

### 修改内容

**文件**：`requirements.txt`

**修改**：在第一行添加 `gunicorn==23.0.0`

```txt
gunicorn==23.0.0
alembic==1.18.4
annotated-doc==0.0.4
...
```

**状态**：
- ✅ 已修改
- ✅ 已提交到 Git
- ✅ 已推送到 GitHub
